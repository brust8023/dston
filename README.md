云演示
==

### 小商城演示访问

由于没有上线，只能在微信开发工具中测试运行：

1. 微信开发工具导入litemall-wx项目;
2. 项目配置，启用“不校验合法域名、web-view（业务域名）、TLS 版本以及 HTTPS 证书”
3. 点击“编译”，即可在微信开发工具预览效果；
4. 也可以点击“预览”，然后手机扫描登陆。

![](doc/pic/5.gif)    

### 管理平台演示访问

1. 浏览器打开，输入以下网址`http://122.152.206.172:8080/#/login`
